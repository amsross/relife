# Conway's Game of Life (in ReasonML)

## Do It
```
npm ci                                # install the dependencies
npm run build                         # build the code
node lib/js/src/Demo.bs.js <pattern>  # run the code
```

## Available Patterns

  * `acorn`
  * `blinker`
  * `diehard`
  * `engine`
  * `glider`
  * `growth`
  * `r_pentomino`
  * `rabbits`
